## Overview: AK (Anna & Kat) Paint
Using HTML Canvas, I will create a basic drawing app with different tools. The minimum viable product is a working canvas with a brush and eraser. 
Will aim to implement features such as brush size, color, and type.

# If Statements and Loops 
Constant while loop to check if pen is down, if so then draw a line, if not then don't draw.

# Event Listeners 
Will listen constantly for pendown vs penup. Will listen for clicks on buttons created in the DOM.

# DOM Element Creation 
Will create buttons and inputs to create canvases, change brushes, and name your work.

# Classes and Subclasses 
Abstract tool class
Brush subclass-- allow for different brush sizes and potentially textures
Eraser subclass

Canvas class-- allows for multiple canvases.
